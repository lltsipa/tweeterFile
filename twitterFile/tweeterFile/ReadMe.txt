This is a console applicatin that is produces a twitter-like feed.  The program reads its data from two files, 
one file contains the users and their interactions such as followers and feeds, the other file contains the 
users posts.  Each user sees posts of who they follow and theirs.  

Instructions:
1. Please use microsoft visual studio to run the application.
2. Open the Program.cs, change the file path (in the "static void Main(string[] args)" function)of the text file 
   to the folder you have installed the program in.
3. Run the program.
